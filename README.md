# PeregrineDB
Peregrine DB is an in-memory thread-safe data store library for Rust.

## Usage: 
In cargo.toml:
```
[dependencies]
peregrine_db = {git = "https://github.com/MarshallBelles/PeregrineDB"}
```

```RS
// Example WIP
```
